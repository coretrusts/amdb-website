快速开始

本指南将帮助您在5分钟内开始使用AmDb数据库。

安装

使用 pip 安装

pip install amdb

从源码安装

git clone https://github.com/coretrusts/amdb.git
cd amdb
python setup.py install

基本使用

创建数据库

from amdb import Database

# 创建数据库实例
db = Database(data_dir='./data/mydb')

# 写入数据
db.put(b'key1', b'value1')
db.put(b'key2', b'value2')

# 读取数据
value = db.get(b'key1')
print(value)  # b'value1'

# 批量写入
items = [(b'key3', b'value3'), (b'key4', b'value4')]
db.batch_put(items)

# 刷新到磁盘
db.flush()

使用CLI工具

# 启动CLI
amdb-cli

# 在CLI中
amdb> connect mydb
amdb [mydb]> put key1 value1
amdb [mydb]> get key1
amdb [mydb]> show keys
