## 搜索引擎提交步骤

### Google Search Console

1. **访问Google Search Console**
   - 网址：https://search.google.com/search-console

2. **添加属性**
   - 点击"添加属性"
   - 选择"网址前缀"
   - 输入：`https://getamdb.com`
   - 验证所有权（推荐使用HTML文件验证）

3. **提交Sitemap**
   - 在左侧菜单选择"Sitemap"
   - 输入：`https://getamdb.com/sitemap.xml`
   - 点击"提交"

4. **请求索引**
   - 在"网址检查"工具中输入首页URL
   - 点击"请求编入索引"

### Bing Webmaster Tools

1. **访问Bing Webmaster Tools**
   - 网址：https://www.bing.com/webmasters

2. **添加网站**
   - 点击"添加网站"
   - 输入：`https://getamdb.com`
   - 验证所有权

3. **提交Sitemap**
   - 在"Sitemap"部分
   - 输入：`https://getamdb.com/sitemap.xml`
   - 点击"提交"

### 百度站长平台

1. **访问百度站长平台**
   - 网址：https://ziyuan.baidu.com

2. **添加网站**

   - 输入：`https://getamdb.com`

3. **提交Sitemap**
   - 在"数据引入" > "链接提交" > "sitemap"

1. **不要过度优化**：避免关键词堆砌

4. **持续更新**：定期更新sitemap和内容
