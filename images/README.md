# 图片资源目录

请在此目录添加以下图片文件：

## 必需文件

- `favicon.ico` - 网站图标（16x16 或 32x32 像素）
- `logo.png` - 网站Logo（建议 200x50 像素）
- `hero-bg.jpg` - 英雄区域背景图（建议 1920x1080 像素）

## 可选文件

- `feature-*.png` - 特性图标（建议 64x64 像素）
- `screenshot-*.png` - 产品截图
- `architecture.png` - 架构图

## 图片优化建议

- 使用 WebP 格式以获得更好的压缩率
- 图片大小控制在 200KB 以内
- 使用适当的图片压缩工具
